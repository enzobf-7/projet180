Détail du webhook

Quand Stripe envoie checkout.session.completed, l'app crée automatiquement :

  • Compte utilisateur Supabase Auth (email + mot de passe temporaire 12 caractères)
  • Profil dans profiles (prénom, nom, email, rôle client)
  • Progression onboarding (5 étapes à compléter)
  • Gamification initialisée (0 XP, niveau 1, streak 0)
  • 2 todos système automatiques (quotidien + dimanche)
  • Email de bienvenue envoyé via Brevo avec les identifiants

Onboarding — 5 étapes

  1. Signature contrat — PDF + champ signature
  2. Questionnaire — 40+ questions, 7 sections
  3. WhatsApp — Rejoindre le groupe
  4. Skool — Rejoindre la communauté
  5. 1er call — Réserver via iClosed Débloqué à l'étape 5

Sécurité

  • Row Level Security (RLS) sur toutes les tables
  • Webhook Stripe vérifié par signature cryptographique
  • Crons protégés par CRON_SECRET
  • Rôle admin vérifié côté serveur (middleware)
  • Mot de passe temporaire sans caractères ambigus
  • Changement de mot de passe disponible dans /profil
